## 学习Markdown的实践

成果展示：

```c
#include <stdio.h>
int main()
{
    int AH,AM,AS,BH,BM,BS,t1,t2;
    while(scanf("%d %d %d %d %d %d",&AH,&AM,&AS,&BH,&BM,&BS)!=EOF)
    t1=(AS+BS)%60;
    t2=(AM+BM+t1/60)%60;
    printf("%d %d %d\n",AH+BH+t2/60,t2,t1);
    return 0;
}

```

对于上式，是用于对2个**时间**的*加法*计算
